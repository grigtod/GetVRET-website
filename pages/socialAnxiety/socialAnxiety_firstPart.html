Supported platforms

This web experience supports both WebXR hardware as well as desktop PC browsers:
- Oculus Quest / Quest 2, Google Cardboard etc
- Desktop mode



How to Play?

To run this web experience you can either click on the clip below or click on this link: presentation.getvret.com



Instructions

WebXR:
  1. Open Oculus Browser
  2. Type webpage: presentation.getvret.com
  3. Wait for the page to load
  4. Tap on the "Enter XR" button on the bottom right
  5. Allow immersive WebXR session by pressing the "Allow" button
Desktop PC:
  1. Open web browser
  2. Visit webpage: presentation.getvret.com
  3. Wait for the page to load
  4. Click on arrows with your mouse to change slides
